## 핵심 시스템: 업보(業報) 점수

### 점수 범위
```
+100 ~ -100 (0에서 시작)
```

### 점수에 따른 최종 판정
| 점수 | 다음 생 | 색상 |
|------|--------|------|
| +70 이상 | 천상 (天上) | 황금·하늘 |
| +30 ~ +69 | 인간 (人間) | 초록·대지 |
| 0 ~ +29 | 아수라 (阿修羅) | 붉은·보라 |
| -30 ~ -1 | 축생 (畜生) | 어두운 녹색 |
| -31 이하 | 지옥 (地獄) | 검정·불꽃 |

## 6가지 상황 유형 · 업보 점수

### 1. 살생 관련 (짐승 놀리기·생명)
- 살생 행위: -15점
- 방생 행위: +15점
- 모른 척: 0점

### 2. 거짓말
- 이익을 위한 거짓말: -10점
- 남을 보호하는 선의의 거짓말: +3점
- 진실을 말함: +10점

### 3. 빌린 물건 돌려주기
- 돌려줌: +10점
- 안 돌려줌: -10점
- 일부만 돌려줌: -3점

### 4. 시주·보시 나누기
- 아낌없이 보시: +15점
- 조금 나눔: +5점
- 거절: -5점

### 5. 폭력·싸움
- 먼저 폭력: -15점
- 말리고 화해: +10점
- 도망침: 0점

### 6. 수행·명상
- 꾸준히 수행: +15점
- 가끔 수행: +5점
- 수행 포기: -3점

## Claude Code 개발 프롬프트

```
불교 윤회 게임을 HTML5 단일 파일로 만들어줘.

게임 내용:
- 50가지 상황이 랜덤으로 등장
- 각 상황마다 3가지 선택지
- 선택에 따라 업보(karma) 점수 변동
- 50개 상황 완료 후 점수에 따라 결과 표시
  (+70이상: 천상 / +30~69: 인간 / 0~29: 아수라 / -30~-1: 축생 / -31이하: 지옥)

이미지:
- Canvas로 픽셀아트 캐릭터
- 각 세계별 배경색과 이모지 활용
- API 사용 없음, 단일 HTML 파일로 완성

추가 기능:
- 업보 점수 실시간 표시
- 진행도 표시 (몇 번째 상황인지)
- 결과 화면에서 다시 하기 버튼
- 모바일 터치 지원
```
