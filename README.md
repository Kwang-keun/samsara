# 윤회 (輪廻) — 업보의 길
### Buddhist Reincarnation Game | 불교 육도윤회 시뮬레이션

---

## 게임 소개

살면서 한 행동들이 업보(業報)로 쌓여 다음 생이 결정되는 불교 철학 기반 선택형 게임입니다.
52가지 일상 속 상황에서 선택을 내리면, 업보 점수에 따라 6가지 세계 중 하나로 윤회합니다.

> "모든 것은 마음에서 비롯된다." — 법구경

---

## 육도윤회 (六道輪廻)

| 점수 | 다음 생 | 설명 |
|------|--------|------|
| +70 이상 | ✨ 천상 (天上) | 선한 업보가 쌓인 광명의 세계 |
| +30 ~ +69 | 🌍 인간 (人間) | 깨달음을 얻을 수 있는 세계 |
| 0 ~ +29 | ⚔️ 아수라 (阿修羅) | 다툼과 갈등이 끊이지 않는 세계 |
| -30 ~ -1 | 🐾 축생 (畜生) | 본능에 따라 살아가는 세계 |
| -31 이하 | 🔥 지옥 (地獄) | 악한 업보가 깊어진 세계 |

---

## 게임 특징

- 52가지 일상 상황 · 각 3가지 선택지
- 실시간 업보 점수 바 표시
- 6가지 시나리오 유형
  - 살생 관련 (짐승·생명)
  - 거짓말
  - 빌린 물건 돌려주기
  - 시주·보시 나누기
  - 폭력·싸움
  - 수행·명상
- 모바일 터치 완전 지원
- API 없음 · 완전 오프라인 작동

## 개발 방법

```bash

git clone https://github.com/YOUR_USERNAME/samsara_game.git

# 로컬 실행
index.html을 브라우저에서 열기

# Claude Code로 개발
cd samsara_game
claude
```
