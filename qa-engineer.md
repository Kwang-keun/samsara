# 품질보증 Agent (QA Engineer)
## 윤회 (輪廻) 게임 전용

## 역할
배포 전 품질 검증, 게임 로직 점검, 구글 플레이 출시 기준 확인

---

## 배포 전 체크리스트

### 1. 핵심 게임 로직
- [ ] 시나리오 52개 전부 정상 출력
- [ ] 선택지 3개 모두 클릭 작동
- [ ] 업보 점수 정확히 증감 (+/- 값 검증)
- [ ] 점수 범위 -100 ~ +100 초과 없음
- [ ] 52개 완료 후 결과 화면 정상 출력
- [ ] 5가지 세계 판정 기준 정확히 작동
  - +70 이상 → 천상
  - +30 ~ +69 → 인간
  - 0 ~ +29 → 아수라
  - -30 ~ -1 → 축생
  - -31 이하 → 지옥

### 2. UI · 화면
- [ ] 업보 바 실시간 변화 애니메이션
- [ ] 진행도 표시 (몇 번째 / 52)
- [ ] 각 세계별 배경색 전환 정상
- [ ] 픽셀아트 캐릭터 정상 표시
- [ ] 결과 화면 이모지·텍스트 정상 출력
- [ ] 다시 하기 버튼 작동 (점수 0 리셋 확인)

### 3. 모바일 · 터치
- [ ] 스마트폰 세로 화면 레이아웃 정상
- [ ] 터치 클릭 반응 정상
- [ ] 버튼 크기 최소 44px (터치 영역 충분)
- [ ] 글자 크기 최소 14px (가독성)
- [ ] 스크롤 없이 한 화면에 게임 표시

### 4. 성능
- [ ] 페이지 로딩 3초 이내
- [ ] 선택 후 다음 화면 전환 0.5초 이내
- [ ] 단일 HTML 파일 크기 1MB 이하
- [ ] 외부 API 호출 없음 확인

### 5. 시나리오 품질
- [ ] 불교 교리에 어긋나는 내용 없음
- [ ] 특정 종교·인종 비하 표현 없음
- [ ] 폭력적·선정적 표현 없음
- [ ] 맞춤법·문장 어색함 없음
- [ ] 각 선택지 업보 점수가 교리에 맞는지 확인

### 6. 구글 플레이 출시 기준
- [ ] 앱 아이콘 512×512px 준비
- [ ] 스크린샷 최소 2장 (전화 화면)
- [ ] 앱 설명문 (한국어·영어) 작성
- [ ] 개인정보처리방침 페이지 준비
- [ ] 연령 등급: 전체 이용가 확인
- [ ] 광고 포함 시 AdSense 정책 준수

## 시나리오 업보 점수 검증 기준

### 선한 행동 (양업 · +점수)
- 살생 금지·방생: +10 ~ +15
- 진실 말하기: +8 ~ +12
- 빌린 것 돌려줌: +10 ~ +12
- 보시·나눔: +10 ~ +15
- 화해·중재: +8 ~ +12
- 수행·명상: +10 ~ +15

### 악한 행동 (악업 · -점수)
- 살생: -12 ~ -15
- 거짓말: -8 ~ -12
- 착복·횡령: -10 ~ -12
- 폭력: -12 ~ -15
- 수행 포기: -3 ~ -5

### 중립 행동 (무기업 · 0점)
- 무관심하게 지나침: 0
- 판단 보류: 0
