# 윤회 (輪廻) 게임 공통 규칙 (CLAUDE.md)

## 프로젝트 개요
- 게임명: 윤회 (輪廻) — 업보의 길
- 플랫폼: HTML5 단일 파일 → Android APK
- 테마: 불교 육도윤회 (六道輪廻)
- 개발: Claude Code (API 없음, 순수 HTML/CSS/JS)

## 모든 Agent 공통 규칙

### 불교 교리 원칙
- 특정 종파 편향 없이 불교 일반 교리 기준
- 살생·거짓·도둑질·음행·음주 = 5계 기준 적용
- 보시·지계·인욕·정진·선정·지혜 = 6바라밀 기준
- 혐오·비하 표현 절대 금지

### 기술 원칙
- 단일 HTML 파일로 완성 (외부 파일 최소화)
- API 호출 없음 (완전 오프라인 작동)
- 모바일 터치 우선 설계
- 파일 크기 1MB 이하 유지

### 게임 데이터 원칙
- 시나리오 총 52개 (6가지 유형)
- 업보 점수 범위: -100 ~ +100
- 선택지: 항상 3개
- 결과: 5가지 세계 (천상/인간/아수라/축생/지옥)

### Git 규칙
- 커밋 메시지: 한국어로 작성
- 배포 전 qa-engineer.md 체크리스트 필수 확인
